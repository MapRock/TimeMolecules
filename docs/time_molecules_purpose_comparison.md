# Table Comparing Time Molecules to Adjacent Technologies

What follows is not a comparison of identical technologies, but of neighboring layers in the same broader stack. These are parts of the world that sit downstream of raw event detection and capture. Once events exist as durable signals, different technologies take different roles: workflow systems orchestrate action, process mining reconstructs and evaluates process behavior, and OLAP platforms organize aggregated analysis. Time Molecules lives alongside these adjacent technologies, overlapping with each in some ways while serving a distinct time-oriented analytical role.

Here I use Temporal Technologies as my concrete example of the workflow orchestration world, process mining as the example of technologies focused on discovering and improving processes from event logs, and Kyvos as the example of the OLAP / semantic layer world. That does not mean each of these exhausts its category. It simply gives the comparison some solid footing, so the reader can see more clearly where Time Molecules fits and where it does not.
