* Prompt: How can I discover hidden relationships between different processes by finding events from separate CaseSets that bump into each other in location or meaning?
* Abstract: The compare_event_proximities.py script processes events from compare_event_proximities.csv that belong to different CaseSets. It detects event collisions by calculating (1) semantic similarity between event descriptions using LLM embeddings stored in Qdrant and (2) geographic proximity using the Haversine formula on latitude and longitude. It then ranks the event pairs by combined score and exports the results to compare_event_proximities_results.csv, including similarity scores, distances, and proposed case link suggestions to help identify related processes.
* Primary location of source material to analyze (for more information): https://github.com/MapRock/TimeMolecules/tree/main/tutorials/link_cases/compare_event_proximities
